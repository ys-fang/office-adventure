# 伺服器機房

推開厚重的門，冷氣迎面撲來，和外面忙碌的辦公區簡直是兩個世界。機房裡的伺服器們排列成行，指示燈不停閃爍，發出永不停歇的嗡嗡聲。其中一台機器上貼著大大的「禁止觸碰」標籤，字體被加粗了三次。角落裡居然有一個睡袋和一個枕頭，看來曾經有人把這裡當成了避難所。

## 可以做的事

碰一下那台「禁止觸碰」的機器：你伸出手指輕輕碰了一下機殼。什麼都沒發生。你再碰一下。還是什麼都沒發生。你開始懷疑那個標籤只是為了嚇阻好奇的人。但就在你轉身的瞬間，所有指示燈同時閃了一下。大概只是巧合吧。
查看伺服器的監控螢幕：角落有一台小螢幕，上面跑著各種圖表和數字。CPU 使用率、記憶體、網路流量——一切看起來都很正常。但你注意到其中一台伺服器的名稱叫做「拜託不要掛」，另一台叫做「備用的備用」。工程師的命名品味一覽無遺。*伺服器名牌*
鑽進角落的睡袋試躺看看：你拉開睡袋的拉鍊，裡面還殘留著體溫（不對，那是伺服器的散熱）。睡袋旁邊有一本翻到一半的技術書籍和兩罐喝完的能量飲料。牆上用便利貼排列出一行字：「活著就是一種部署」。
