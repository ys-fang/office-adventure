# 工程師角落

辦公區最深處，三面螢幕同時亮著不同的視窗。桌上散落著空的能量飲料罐和吃了一半的洋芋片，隔壁的工程師正戴著耳機專注地打字。機械鍵盤旁邊坐著一隻黃色小鴨——工程師們的精神導師。便利貼上寫著各種令人費解的訊息：「不要動 line 847」、「這個 function 沒有人理解，包括我」。

## 可以做的事

在鍵盤上隨便打幾個字：你坐下來，手指碰上機械鍵盤。清脆的打字聲在深夜的辦公室裡格外響亮。螢幕上的終端機還開著，你打了一行「hello」然後按下 Enter。系統回應：「command not found」。你和這台電腦之間的溝通也跟平常的 code review 一樣順暢。
拿起小鴨研究它的用途：這是一隻經典的除錯用橡皮小鴨。根據傳說，對著它解釋你的程式碼，就能找到 bug。鴨子的底部用奇異筆寫著歷任主人的名字，已經傳了四手。鴨子的表情永遠是那種微笑——無論你的程式碼有多糟糕，它都不會評判你。*除錯小鴨*
試著讀懂螢幕上的程式碼：你盯著螢幕上密密麻麻的程式碼看了三十秒。這段程式碼的註解寫著「暫時解法，之後會改」，根據 git log 的日期，這個「暫時」已經持續了兩年。你佩服這段程式碼的生命力，默默關掉了視窗。
幫忙清理桌上的空罐子：你把空的能量飲料罐和洋芋片包裝收集起來，整理到旁邊的垃圾桶。桌面清爽了不少，露出了底下壓著的一張小紙條：「如果你正在幫我收拾桌子，謝謝。冰箱裡第二層左邊的布丁請自取。——坐這裡的人」。
