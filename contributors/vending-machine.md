# 販賣機角落

走廊盡頭有兩台販賣機，一台賣飲料、一台賣零食。午後這裡是同事們補充能量的熱門據點，不時有人走過來投幣。零食機裡有一包洋芋片卡在出口半天掉不下來，看起來已經維持這個姿勢很久了。投幣口旁邊用膠帶貼著一堆硬幣兌換的小紙條。

## 可以做的事

用力拍一下零食販賣機看能不能把卡住的洋芋片震下來：你對準側面用力拍了一掌。販賣機晃了一下，洋芋片紋風不動。你又拍了一掌，這次更用力。洋芋片微微傾斜，然後又回到原位。它用行動證明了什麼叫做「穩如泰山」。旁邊有人用便利貼記錄著嘗試次數，已經來到第三十七次了。*挑戰紀錄*
研究販賣機咖啡的選項差異：飲料機的選單有八種咖啡：美式、拿鐵、卡布奇諾、摩卡、黑咖啡、特濃、加糖、少糖。你仔細看了看，發現不管選哪一種，出來的液體顏色都差不多。底部有一張用戶評論：「八種選擇，一種味道。但深夜兩點你不會在乎。」你覺得非常有道理。
投一枚硬幣許個願：你掏出一枚十元硬幣，但沒有投進投幣口，而是丟進了販賣機和牆壁之間的縫隙。硬幣叮叮噹噹地滾到了某個看不見的地方。你對著販賣機許願：「讓那包洋芋片掉下來」。什麼都沒發生，但你覺得自己為辦公室的都市傳說增添了一枚硬幣。
