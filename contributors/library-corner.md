# 圖書角

辦公室的一個角落被改造成了小型圖書館。兩面矮書架圍出一個 L 形的閱讀空間，中間放了一張看起來極度舒適的單人沙發。窗外的光線剛好灑進來，照在書架上那些被翻過無數次的書脊上。書架旁邊掛著一本閱讀紀錄本，但最後一筆紀錄已經是兩個月前。

## 可以做的事

坐進那張單人沙發：你一坐下就明白為什麼沒有人在閱讀紀錄上寫字——因為坐上去就不想動了。沙發的角度剛好可以靠著看書，扶手的寬度剛好可以放一杯咖啡。你懷疑這張沙發是辦公室裡最危險的傢俱，因為它會讓人忘記自己還在上班。
翻閱閱讀紀錄本：紀錄本的格式是「日期 / 書名 / 讀者 / 心得」。大部分心得很正經：「觀點新穎」「值得一讀」。但有幾條比較誠實：「在這裡睡著了，不確定自己讀了什麼」「這本書教會我一件事：我不適合讀這類書」「看了三頁就被叫去開會」。
隨手從書架抽一本書翻開隨機一頁：你閉著眼睛抽了一本。是一本關於時間管理的書。你翻到第 87 頁，上面寫著：「學會說不。你不需要參加每一場會議。」有人用鉛筆在旁邊重重地畫了底線，還加了三個驚嘆號。你覺得這位讀者一定深有感觸。*時間管理書*
在閱讀紀錄本上寫下一筆：你拿起旁邊的筆，在最新的一行寫下今天的日期和「深夜探險，路過翻了幾本書」。你猶豫了一下心得欄，最後寫了：「辦公室深夜比白天安靜，很適合看書，可惜我在探險」。也許下一個翻到這頁的人會覺得有趣。
