# 洗手間

你推開洗手間的門，日光燈嗡嗡作響，其中一盞微微閃爍。鏡子前的洗手台上散落著幾滴水漬，擦手紙桶已經滿出來了。空氣中殘留著洗手乳的柑橘味，外面隱約傳來同事聊天的聲音。隔間門上不知道被誰貼了各種貼紙，最顯眼的那張寫著「你今天也辛苦了」。

## 可以做的事

盯著鏡子裡的自己看十秒鐘：你直視鏡中的自己。那是一張深夜加班的臉，眼睛有點腫，頭髮有點亂。但你對自己點了點頭。鏡子邊框上有人用白板筆寫了一行字：「這裡是全辦公室唯一不用假裝精神很好的地方」。你覺得深夜的鏡子特別誠實。
閱讀隔間門上的塗鴉：你推開第一間隔間的門，門背面是一場跨越數月的對話。「這個 bug 修了三天」下面有人回：「只有三天？你很幸運」。再下面：「我覺得 bug 比我有毅力」。最底下用不同顏色的筆寫著：「各位，這裡是廁所，不是 code review」。*廁所語錄*
把滿出來的擦手紙桶壓一壓看能不能塞更多：你用力往下壓，紙桶裡的擦手紙發出抗議般的聲響。你成功騰出了大約三張的空間。就在你覺得自己為人類做出貢獻的時候，旁邊的感應式擦手紙機突然自動吐出一張紙，像是在說「你需要這個」。
把「你今天也辛苦了」的貼紙端正貼好：那張貼紙已經歪了，邊角翹起來快要掉了。你小心地把它壓平，重新貼正。然後你從口袋裡找到一張空白便利貼，在上面寫了「明天也會沒事的」，貼在旁邊。
